
# 저수율 예측 모델

강릉의 오봉 저수지 데이터를 기반으로 오봉 저수지의 저수율을 예측하는 것을 목표로 하는 프로젝트이다.




## Authors

- [@don-gik](https://www.github.com/don-gik)


## Documentation

### How to run

- ``` git clone https://https://github.com/don-gik/modeling ```
- Python Package Manager [UV](https://docs.astral.sh/uv/)를 설치한다.
- ```modeling``` directory로 가서 ``` uv venv ```명령어를 실행한다.
- ```source .venv/bin/activate``` 혹은 ```.\.venv\Scripts\activate```을 통해 uv virtual environment를 활성화한다.
- ``` uv add -r requirements.txt ```를 통해 requirements를 설치한다.
- ```./pipeline/``` 에 있는 코드에 실행 권한을 부여한 후 실행하고 싶은 스크립트를 실행한다.
